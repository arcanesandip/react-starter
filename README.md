# react-starter

Personal React + Tailwind starter. Clone and build.

## What's inside

- Vite + React
- Tailwind CSS v4
- ESLint (flat config) + Prettier
- Absolute imports (`@/`)
- Folder structure ready to go

## Usage

```bash
npx degit arcanesandip/react-starter my-new-project
cd my-new-project
npm install
npm run dev
```

## Folder structure

```
src/
├── assets/
├── components/
│   └── ui/
├── hooks/
├── layouts/
├── pages/
└── utils/
```
